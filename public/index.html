About This System

Awe-RA Clinic Management System — Prototype v1.0

This is a working prototype that runs entirely in your browser. All data is currently stored locally on this computer (browser storage). To make it accessible from any computer nationwide, please read the Deployment Guide document that was delivered alongside this file.

Features included: client records (shared across branches), appointment scheduling (per-branch calendar), inventory management (per-branch stock), multi-branch support, user roles (admin/reception/therapist), data export/import.

Next steps to go nationwide: host on cloud (Firebase / Supabase recommended), add real authentication, enable real-time sync between branches.
